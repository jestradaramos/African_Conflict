Wordcloud maker
---------------

FILES:
---------------
african_conflicts.csv - Data to be sorted
africa2.png - Image template for wordcloud

Text_sort.py - sorts the .csv file into multiple text files that contains notes for the entire year
Create_wordcloud.py - Creates yearly word clouds as .png based on text files created



HOW TO RUN:
---------------
1. Have all files in the same folder
2. Run Text_sort.py first to create text files to use
3. Run Create_wordcloud.py to create individual word clouds as .png for each year
